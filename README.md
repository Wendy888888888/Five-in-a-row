# Five（五子棋）

基于QT框架开发的五子棋游戏，支持人人对战、人机对战和机机对战模式，提供完整的游戏功能与个性化设置。


## 功能特性

- **多种对战模式**
  - 人对人（PvP）
  - 人对机（PvE）
  - 机对机（观赏模式）
  
- **游戏设置**
  - 选择先手/后手
  - 自定义棋子颜色（黑/白）
  - 两种AI难度（简单/困难）

- **核心功能**
  - 实时胜负判定
  - 无限次悔棋操作
  - 游戏进度保存/加载（.five格式）
  - 步数统计与状态显示

- **界面特性**
  - 高清棋盘渲染
  - 棋子渐变效果
  - 智能坐标映射

## 安装运行

### 环境要求
- Qt 5.15+ 
- C++11 及以上
- CMake 3.5+（可选）
